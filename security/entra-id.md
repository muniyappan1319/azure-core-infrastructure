## Overview
This section documents identity and access management practices implemented
to secure Azure cloud infrastructure.

## Role-Based Access Control (RBAC)
- Applied least-privilege access using Azure built-in roles
- Scoped role assignments at subscription and resource group levels

## Multi-Factor Authentication (MFA)
- Enforced MFA for users and privileged accounts
- Secured access to Azure resources and portals

## Conditional Access
- Configured Conditional Access policies based on risk and location
- Blocked legacy authentication methods

## Outcome
- Improved overall cloud security posture
- Reduced unauthorized access risks
